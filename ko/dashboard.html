대시보드

전략을 한눈에 모니터링하는 데 필요한 모든 것.

전략 모니터

현재 위치와 목표 위치를 정확히 확인하세요. 전략 모니터는 레시피의 목표 배분과 실시간 현재 배분을 함께 보여줍니다.

  • 현재 가격 기준 목표 배분 대비 현재 배분
  • 괴리 폭과 traig가 이를 좁히기 위해 하는 일
  • 실시간 업데이트 — 새로고침 불필요

모든 것을 한 곳에

대시보드는 트레이딩 관리에 필요한 모든 것을 한데 모았습니다.

전략 바로가기

저장된 레시피에 빠르게 접근. 전략 전환, 일시정지, 조정 — 모두 손이 닿는 곳에.

거래소 정보

거래소 데이터를 실시간으로 가져옵니다 — 사용 가능한 잔액, 잠금 금액, 계정 상태를 한눈에.

수동 주문

수동으로 제어하고 싶을 때 대시보드에서 직접 주문을 넣으세요. 거래소로 전환할 필요 없이.

주문 목록

대기 중 및 체결된 모든 주문을 확인. traig가 무엇을 하고 있고 무엇이 실행됐는지 추적하세요.

볼륨 뷰 — 목표 배분 대비 편차(%)로 표시된 내 주문

깊이 뷰 (₩) — 각 가격 레벨의 누적 KRW 깊이

깊이 뷰 (%) — 내 주문 % 오버레이가 포함된 누적 깊이

단순한 오더북이 아닙니다. traig는 라이브 오더북 위에 전략을 오버레이하여 메이커 주문이 시장 대비 정확히 어디에 위치하는지 볼 수 있게 해줍니다.

  • 라이브 오더북 내에서 강조 표시된 내 주문
  • 전략이 시장 깊이와 어떻게 상호작용하는지 확인
  • 시장이 움직이는 대로 실시간 업데이트
